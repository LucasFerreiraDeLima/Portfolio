Projeto: Estação Meteorológica de Custo Reduzido

Explicação rápida sobre o projeto

Projeto de estação meteorológica com custo reduzido, desenvolvido durante o ensino médio/técnico, colocando em prática conhecimentos adquiridos no decorrer do curso.

Em resumo, o projeto trata-se de um conjunto de sensores, circuitos elétricos, um microcontrolador (Arduino), painel solar para alimentação e dispositivo para envio de dados para a internet.

O objetivo central do projeto foi desenvolver uma estação meteorológica com custo reduzido, para auxiliar principalmente agricultores de baixa renda que não podem pagar por tecnologias mais caras.

Foi um projeto de TCC, desenvolvido desde o 1º ano, e apresentado em diversos eventos de tecnologia no estado de São Paulo.

Para mais detalhes, há uma explicação sobre o projeto no Linkedin (link para o meu perfil na seção Contato desse site)
